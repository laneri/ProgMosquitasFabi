Programa mosquitas2020.f90 (adaptado con nueva rutina de números aleatorios ran2) de FabianaL pasado a lenguaje C por AnaG.

1) Para compilar y correr el programa, descargar archivo ran2.h y programa_3.c,
ejecutar en la consola ingresando
gcc -o programa programa_3.c -lm 
./programa

2) La salida de datos tienen extensión .dat
3) En la pantalla de la consola se imprimen las condiciones iniciales para las mosquitas.
